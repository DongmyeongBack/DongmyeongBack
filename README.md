## Featured Projects

### 🌀 [ryoiki-tenkai](https://github.com/DongmyeongBack/ryoiki-tenkai)
> Real-time hand gesture recognition in the browser, inspired by *Jujutsu Kaisen*

Detects domain expansion poses (Gojo, Megumi, Ryomen) via webcam using MediaPipe for hand landmark detection and a Keras MLP model converted to TensorFlow.js for in-browser inference. Visual effects are rendered with the Canvas API.

`Vite` `JavaScript` `TensorFlow.js` `MediaPipe` `Keras` `Python`

---

### 🛒 [HODU Open Market](https://github.com/modulab-FE6-team5/openMarket)
> Full-featured e-commerce platform built with vanilla JS

A multi-page marketplace supporting buyer/seller flows — product listing, cart management, and checkout — built with pure HTML/CSS/JS and Vite. Includes debounced search, role-based UI, and inventory management.

`Vite` `JavaScript` `HTML5` `CSS3`

---

### 🍊 [Mandarin Market](https://github.com/Hallabong-Frontend/mandarin-market)
> Social commerce app built with React + Vite

A collaborative frontend project with a React SPA architecture, enforcing code quality via ESLint and Prettier.

`React` `Vite` `JavaScript`
